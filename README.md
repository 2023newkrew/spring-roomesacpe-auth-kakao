# spring-roomescape-auth

- [X] 토큰 발급하는 API 생성
  - [X] username으로 사용자를 조회하는 기능
  - [x] 등록된 사용자의 password와 요청으로 넘어온 password가 일치하는지 확인하는 기능
  - [X] 사용자에게 accessToken을 발급하는 기능
- [x] 내 정보 조회하기
  - [x] 유효한 토큰인지 검증하는 기능
  - [x] 토큰을 이용하여 사용자의 정보를 조회하는 기능
- [x] 예약하기 API 개선
  - [x] 비로그인 사용자는 예약이 불가능하다.
- [x] 예약취소 API 개선
